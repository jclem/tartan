tartan

tartan is a simple, flexible grid system packaged as a Ruby gem.

Install

Install the gem:

$ gem install tartan-grid

Install tartan SCSS:

$ tartan install --path app/assets/stylesheets

Or bundle the gem:

$ echo "gem 'tartan'" > Gemfile
$ bundle

Usage

Create a container element:

<div id='content'></div>
@import 'tartan/tartan';

#content {
  @include tartan-container;
}

Full-width content uses tartan-span-columns, partial-width content uses tartan-span-columns($column-count). By default, a container has 12 columns, so tartan-span-columns(6) is a 50%-width section.

<div id='content'>
  <header>
    <h1>Buy This Thing</h1>
  </header>

  <section id='pricing'>
    <div class='pricing-plan'>Pricing plan 1</div>
    <div class='pricing-plan'>Pricing plan 2</div>
  </section>
</div>
@import 'tartan/tartan';

#content {
  @include tartan-container;
}

header {
  @include tartan-span-columns;
}

.pricing-plan {
  @include tartan-span-columns(6);
  &:last-child { @include tartan-omega; }
}

For nested content, wrap content to be nested in tartan-sub-container($context). Further nested content will make use of the same $context: 

<div id='content'>
  <header>
    <h1>Buy This Thing</h1>
  </header>

  <section id='pricing'>
    <div class='pricing-plan'>Pricing plan 1</div>
    <div class='pricing-plan'>Pricing plan 2</div>
  </section>

  <aside>
    <h2>Hello from Sidebar</h2>
  </aside>

  <div id='primary-content'>
    <h2>Features</h2>

    <ul class='feature-list'>
      <li>Stuff you don't want</li>
      <li>Stuff you don't need</li>
    </ul>

    <ul class='feature-list'>
      <li>Stuff you don't want</li>
      <li>Stuff you don't need</li>
    </ul>
  </div>
</div>
@import 'tartan/tartan';

#content {
  @include tartan-container;
}

header {
  @include tartan-span-columns;
}

.pricing-plan {
  @include tartan-span-columns(6);
  &:last-child { @include tartan-omega; }
}

aside {
  @include tartan-span-columns(4);
}

#primary-content {
  @include tartan-sub-container(8);
  @include tartan-omega;

  h2 { @include tartan-span-columns(8, 8); }
}

.feature-list {
  @include tartan-span-columns(4, 8);
  &:last-child { @include tartan-omega; }
}
